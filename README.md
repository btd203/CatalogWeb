# Product Catalog Website

A modern product catalog website built with Next.js 14+, TypeScript, Tailwind CSS, and Sanity.io CMS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.local.example .env.local
```

3. Update `.env.local` with your Sanity.io credentials and site configuration.

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript (Strict mode)
- **Styling:** Tailwind CSS (Mobile-first)
- **CMS:** Sanity.io
- **Icons:** Lucide React

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - React components (UI and modules)
- `lib/` - Utilities and helpers
- `sanity/` - Sanity configuration and schemas
- `types/` - TypeScript type definitions
